# BRAYN v0.15

Этот репозиторий теперь запускает и деплоит **новый прототип `v0.15`**.
Старый root-прототип больше не используется как точка входа.

## Локальный запуск

Откройте файл:

- `v0.15/index.html`

или поднимите локальный сервер из корня репозитория:

```bash
python3 -m http.server 4173 --directory v0.15
```

После этого откройте:

- `http://localhost:4173`

## GitHub Pages

В корне сайта настроен переход на новый прототип:

- `https://olehhebel.github.io/BRAYN/` → `https://olehhebel.github.io/BRAYN/v0.15/`

## Структура

- `v0.15/` — актуальный прототип (HTML/CSS/JS)
- `index.html` — redirect на `v0.15`
