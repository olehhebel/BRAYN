  • Reward (S7) → pure celebration, earns 300 Neurons + 2 Minutes

Developer / MCP Handoff Notes

  • Terminology: "Sprint" → "Phase" in visible UI. "Nanoframework" → "Boost". First task = full Boost.
  • Screen 6 rule: Time answer is only a nominal signal for future notification setup. It does NOT define phase depth.
  • Coach routing: Determined by Screen 5 (goal) selection. Kayra = 4 goals, Orra = 1 goal, Maverick = 1 goal.
  • Kayra sub-routing: Each of 4 Kayra goals maps to a different path title, boost, and growth priority.
  • Input validation: BRAYN name CTA active only after 2+ characters entered.
  • Bubble select: Single-select only — selecting a new bubble deselects the previous one.
  • Splash: 3-second auto-advance. No user interaction required.
  • Navigation: Back button on all non-reward/celebration screens. No dead ends. Single primary CTA per screen.
  • Auto-layout: All screens use flex column layout with safe-area padding. Mobile-first. Max width 420px.
  • Colors: Kayra #00DA30, Orra #00FFFF, Maverick #FE6305. Accent #A855F7. Background #0A0B14.
  • Prototype is live at: /docs/onboarding-v015/ on GitHub Pages.
