# Silver Pulse delete account prototype

Self-contained inline HTML prototype for the 3-state Silver Pulse iOS flow:

- Base profile/settings screen
- Settings dropdown anchored to the top-right action button
- Delete-account confirmation popup layered above the menu state

## Files

- `index.html` — single-file prototype with inline CSS and inline JavaScript
